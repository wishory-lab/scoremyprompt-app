## How to Convert SVG → PNG

### Option 1: Browser (easiest)
1. Open the SVG file in Chrome
2. Right-click → "Save image as..." → Save as PNG
3. For exact pixel sizes, use browser zoom 100%

### Option 2: Online Converter
- https://svgtopng.com — drag & drop, set dimensions

### Option 3: Command Line (if you have Inkscape)
```bash
inkscape profile-512.svg -w 512 -h 512 -o profile-512.png
inkscape x-header-1500x500.svg -w 1500 -h 500 -o x-header-1500x500.png
inkscape linkedin-banner-1128x191.svg -w 1128 -h 191 -o linkedin-banner-1128x191.png
inkscape youtube-art-2560x1440.svg -w 2560 -h 1440 -o youtube-art-2560x1440.png
```
