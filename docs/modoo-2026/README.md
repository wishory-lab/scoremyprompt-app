## 📋 핵심 사실 (검증된 정보)

### 제품 현황
- **Domain**: https://scoremyprompt.app
- **Stack**: Next.js 14 + TypeScript + Tailwind + Supabase + Claude API + Stripe
- **Deployed**: Production 운영 중 (영어 글로벌 먼저 런칭)
- **Framework**: PROMPT 6차원 평가 엔진
  - P — Precision (정밀도)
  - R — Role (역할)
  - O — Output Format (출력 형식)
  - M — Mission Context (미션 컨텍스트)
  - P — Patterning (구조)
  - T — Tailoring (맞춤화)
- **Tiers**:
  - Free: 매일 10건, 회원가입 불필요
  - Pro: $9.99/월 (7일 무료 체험)
- **Pro Features**: 무제한 평가, AI 리라이팅, 히스토리, Bulk 5개 동시, HTML 리포트
- **Job Tracks**: Marketing, Design, Product, Finance, Freelance, Engineering
- **PWA**: 모바일 홈화면 설치 지원
- **Share**: X, LinkedIn, Copy, Download
- **Scoring**: 0~100점 + S/A/B/C/D 등급 + 피어 퍼센타일
