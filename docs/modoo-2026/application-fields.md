## [FIELD 4] 해결 방안 (아이디어 내용)

```
ScoreMyPrompt는 프롬프트를 붙여넣으면 5초 안에 100점 만점 점수와 상세 피드백, 개선안을 제시하는 AI 프롬프트 품질 관리 플랫폼이다. 업계 최초의 PROMPT 6차원 평가 프레임워크로 평가한다.

▸ P — Precision (정밀도): 목표·대상·범위 구체성
▸ R — Role (역할): AI에게 부여한 역할·전문성
▸ O — Output Format (출력 형식): 결과물 형태 명확성
▸ M — Mission Context (미션 컨텍스트): 배경·상황·제약조건
▸ P — Patterning (구조): 논리적 순서·섹션 구분
▸ T — Tailoring (맞춤화): 톤·독자·사용 맥락 반영

핵심 기능은 모두 프로덕션에서 작동 중이다. 즉시 평가(0~100점 + S/A/B/C/D 등급), 차원별 개선 제안("P차원 6/20점 → 이 부분을 이렇게 바꾸면 15점 상승"), Marketing·Design·Product·Finance·Freelance·Engineering 6개 직무 트랙, Pro 기능(무제한 평가·AI 자동 리라이팅·히스토리·Bulk 5개 동시 평가·HTML 리포트), PWA 모바일 앱 설치, 소셜 공유(X·LinkedIn)로 바이럴 루프를 설계했다.

심사위원님께서는 지금 scoremyprompt.app에 접속하시면 본 아이디어를 회원가입 없이 즉시 체험하실 수 있습니다. 매일 10건 무료.
```
