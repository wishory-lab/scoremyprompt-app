# 숏폼 영상 스크립트 (30~60초)

## 목표
- 길이: 45초 권장
- 포맷: YouTube Shorts (9:16 세로), 완성본 URL 제출
- 목적: 심사위원에게 "실제 작동 증명"

## 스토리보드

| 시간 | 화면 | 자막/내레이션 |
|------|------|---------------|
| 0~3s | ChatGPT 실망 답변 | "AI한테 물어봤는데… 별로네" |
| 3~8s | scoremyprompt.app 접속 | "프롬프트가 문제였습니다" |
| 8~18s | 한국어 프롬프트 입력 → 5초 평가 | "PROMPT 6차원 평가" |
| 18~28s | D등급 42점 → 개선 제안 | "정확히 뭐가 부족한지 알려드립니다" |
| 28~38s | 수정 프롬프트 → A등급 85점 | "같은 AI, 10배 나은 결과" |
| 38~45s | 로고 + 엔딩 카드 | "ScoreMyPrompt — 부산에서 시작한 한국 최초 AI 프롬프트 품질 도구" |

## 제작 도구

- CapCut 또는 Canva Video
- Luke 기존 도파민 숏폼 워크플로우 재사용 가능

## 업로드

- YouTube Shorts (공개, 조회수 무관)
- URL을 modoo.or.kr 도전신청서에 첨부

## 촬영 체크리스트

- [ ] ChatGPT 실망 답변 스크린샷 확보 (도입부 3초용)
- [ ] scoremyprompt.app 한국어 프롬프트 샘플 2개 준비
  - 예: "모두의창업 지원서 기획 도와줘" (D등급 예상)
  - 예: "당신은 정부 지원사업 기획 전문가입니다. 모두의창업 신속심사에 지원할 ScoreMyPrompt(AI 프롬프트 품질 평가 SaaS)의 문제인식 필드를 500자 이내로 작성하세요. 포맷: 3문단, 첫 문단은 한 줄 요약." (A등급 예상)
- [ ] 화면 녹화: OBS / QuickTime / Mac Screen Record
- [ ] BGM: Epidemic Sound 또는 YouTube Audio Library 저작권 무료
- [ ] 엔딩 카드: scoremyprompt.app 도메인 + Luke 이메일

## 제출 전 최종 확인

- [ ] 45초 이하
- [ ] 세로 9:16 해상도 (1080×1920 이상)
- [ ] 자막 삽입됨 (음소거 시청 대응)
- [ ] 허위 수치 포함된 랜딩 화면 아닌 — 교체본 사용
- [ ] YouTube Shorts URL이 modoo.or.kr 폼에 정상 붙여넣어짐
