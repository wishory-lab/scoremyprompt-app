## 네이버 블로그 발행 방법

1. **로컬에서 미리보기**: 위 HTML 파일을 더블클릭 → 브라우저에서 열어 확인
2. **HTML 소스 복사**:
   - 메모장/VS Code로 열기
   - `<body>` 태그 안의 모든 내용을 복사 (또는 전체 복사)
3. **네이버 블로그 글쓰기**:
   - 글쓰기 → 우측 상단 **"HTML"** 버튼 클릭 (Smart Editor → HTML 모드)
   - 복사한 코드 붙여넣기
   - 다시 일반 모드로 전환 → 미리보기 확인
4. **이미지 추가**:
   - HTML에는 이미지가 없음 (인용 박스로 대체)
   - 별도 캡처/생성한 이미지 5장 정도 본문에 직접 삽입 권장
   - 위치: 도입부 1장(썸네일), PROMPT 6차원 1장, ScoreMyPrompt 화면 1장, 점수 비교 1장, 글로벌 비전 1장
5. **태그 입력**: 본문 맨 아래 해시태그 25개를 발행 시 태그란에 한 번 더 입력

## HTML 호환성

- 네이버 블로그 Smart Editor는 인라인 `style=""`를 거의 다 보존
- `<script>`, 외부 `<style>` 태그는 자동 제거됨 (안전)
- `<table>`, `<blockquote>`, `<a>`, `<hr>`, `<strong>` 모두 지원
- 단색 background, border, border-radius, padding, margin 모두 OK
